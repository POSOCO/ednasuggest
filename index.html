Instructions

  1. Click the choose files button and then upload the monitoring logic files
  2. Click Add logic files button
  3. Click Create functions button
  4. Click Start monitoring button to start getting suggestions from the app
  5. Click Stop monitoring button to stop suggestions from the app
  6. Click Clear logic files button to clear the existing monitoring logics
